# 💼 Job Tracker

İş başvurularını düzenli takip etmek için geliştirilmiş bir web uygulaması.

## 🖼️ Ekran Görüntüsü

![Job Tracker 1](./screenshot1.png)
![Job Tracker 2](./screenshot2.png)

## 🚀 Özellikler

- ➕ Yeni başvuru ekleme
- 📋 Başvuruları listeleme
- ✏️ Başvuru güncelleme
- 🗑️ Başvuru silme
- 🔍 Şirket ve pozisyona göre arama
- 📊 Durum bazlı filtreleme
- 📈 İstatistik paneli

## 🛠️ Teknolojiler

- React
- Tailwind CSS
- LocalStorage
- Vite
